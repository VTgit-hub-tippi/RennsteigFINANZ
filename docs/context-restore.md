# Kontext-Restore (Arbeitsanweisung)

Wenn eine neue KI-Session gestartet wird:

1. Lies zuerst STATUS.md, NEXT-ACTION.md, CLAUDE.md und docs/context-snapshot.md.
2. Verifiziere Git-Zustand (Branch, saubere Arbeitskopie, Ahead/Behind).
3. Rekonstruiere den aktuellen Arbeitsfokus in maximal 8 Punkten.
4. Nenne genau eine naechste ausfuehrbare P0-Aktion.
5. Markiere Unsicherheiten explizit als OFFEN statt zu raten.

Ziel: sofortige Wiederaufnahme ohne Kontextverlust.
